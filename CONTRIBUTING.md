# Contributing to Epstify

Thanks for your interest in contributing! 🔲

## How to contribute

1. Fork the repo
2. Create a branch: `git checkout -b feature/your-feature`
3. Make your changes to `index.html`
4. Test locally by opening `index.html` in your browser
5. Commit: `git commit -m "feat: describe your change"`
6. Push: `git push origin feature/your-feature`
7. Open a Pull Request

## Ideas for contributions

- New stamp types (CIA, NSA, PENTAGON, etc.)
- New easter egg triggers
- Better PDF multi-page support
- Improved mobile UX
- New redaction animation effects
- Telegram bot integration

## Rules

- Keep it 100% client-side — no backend, no APIs, no tracking
- Keep it in a single `index.html` file
- Keep it fun

## Reporting bugs

Open an issue with:
- What you did
- What you expected
- What happened instead
- Browser + OS
